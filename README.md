# 🐾 DJ Tiger Paws — OpenClaw AI Agent

## 🧠 Overview

DJ Tiger Paws is a **persistent, multi-channel AI agent** built on OpenClaw, designed to operate with personality, memory, and real-world utility.

Publicly, it presents as a chaotic, lovable basement DJ dog.  
Privately, it operates as a disciplined system for **automation, decision support, and revenue-focused workflows**.

This project demonstrates how to build a production-ready AI agent with:

- Persistent memory (write + recall)
- Multi-channel interaction (Telegram, Discord, Web UI)
- Secure operational boundaries
- Automated system maintenance (heartbeat + cron)
- Tool integrations (Google CLI, web, filesystem)

---

## ⚡ Key Features

### 🧠 Persistent Memory
- Long-term memory via `MEMORY.md`
- Daily logs via `memory/YYYY-MM-DD.md`
- Automatic memory writes before context compaction
- Cross-session recall (memory + session search)

### 🐾 Persona + Identity Layer
- Defined in `SOUL.md`, `IDENTITY.md`, `USER.md`
- Stable voice and behavioral alignment
- Public vs private role separation

### 🔁 Automation (Heartbeat + Cron)
- `HEARTBEAT.md` maintains system health
- Detects drift, stale tasks, and inconsistencies
- Scheduled workflows (audits, standups, checks)

### 🔐 Security-First Design
- No installs or spending without explicit approval
- Sensitive data never exposed (keys, wallets, credentials)
- Prompt injection awareness

### 🔌 Multi-Channel Operation
- Telegram (DM + commands)
- Discord (server + bot)
- Web UI (OpenClaw Control Panel)

### 🧰 Tooling
- OpenAI Codex (primary via OAuth)
- OpenRouter (fallback models)
- Google CLI (`gogcli`) for Docs, Sheets, Gmail
- Web + filesystem access via OpenClaw tools
